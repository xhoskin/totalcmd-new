Password Cracker - программа для восстановления забытых паролей, скрывающихся за астерисками (символами вида "***").

Для восстановления пароля запустите Password Cracker и наведите курсор мыши на строку с паролем. Пароль будет отображен в окне Password Cracker и\или будет показан непосредственно в окне "восстанавливаемой" программы.

Установка Password Cracker

Password Cracker работает под управлением Windows NT4\95\98\ME\2000\XP\Vista. Password Cracker не требует установки для своей работы. Распакуйте все файлы из zip-архива, и запустите файл pwdcrack.exe.

Настройки

Восстанавливать в Internet Explorer
Восстанавливать пароль в окнах Internet Explorer. Для восстановления пароля активизируйте на веб странице элемент ввода пароля (т.е. просто кликните по нему мышкой). Восстановленный пароль отображается только в окне Password Cracker.
Примечание: для восстановления пароля необходим установленный Internet Explorer 5.0 или старше.
Восстанавливать во всех окнах
Password Cracker проверяет может ли элемент управления под курсором содержать пароль. И восстанавливает пароль только в тех элементах управления, которые по мнению программы оный пароль содержать могут, прочие же элементы управления игнорируются. При включенной опции Password Cracker будет пытаться восстановить пароль во всех элементах управления, вне зависимости может ли там содержаться пароль или нет.
Поверх всех
Всегда располагать окно Password Cracker поверх всех окон.

Известные проблемы

Некоторые из антивирусов идентифицируют файл pchook.dll как вирус или "троянский конь", хотя на самом деле это не так.
Такая точка зрения аргументируется тем, что программа может быть использована злоумышленниками для получения пароля. Чисто теоретически, конечно, можно использовать Password Cracker в подобных целях, если Вы и есть этот самый пресловутый злоумышленник. Антивирусы, как правило, блокируют доступ к файлу pchook.dll, а без этого файла Password Cracker работать не будет.

Автор многократно обращался в одну из известных Российских антивирусных компаний, и дважды, выслушав мои аргументы, они исключали pchook.dll из вирусных списков. Некоторое время назад я с удивлением снова обнаружил pchook.dll в их вирусном списке - видимо процесс включения\исключения итеративен по своей природе :). Я намеренно не привожу название компании, чтобы не создавать им рекламы, тем более такой сомнительной.
Что можно посоветовать:

  1. Попробуйте отключить антивирус.
  2. Большинство антивирусных компаний предлагают проверку Ваших файлов на вирусы в online. Проверьте Password Cracker. И если будет что-либо обнаружено, внимательно прочитайте что именно. Частенько антивирусы сообщают кратко "Вирус..." или "Инфицирован...", а в более развернутом описании уже можно встретить фразы более нейтрального тона, что мол "вообще-то и не очень вирус, но сугубо теоретически если б у дедушки были бы ..., тогда можно использовать..." и так далее, и в том же духе.
    Пожалуйста, прочитайте внимательно и убедитесь сами, чем же именно якобы опасен незабвенный файл pchook.dll.
  3. И наконец, можно не "забывать" пароли, тогда надобности в Password Cracker попросту не возникнет, и можно смело, безо всякой оглядки выполнять абсолютно все рекомендации подобного антивирусного программного обеспечения. Возможно такой способ не всем подходит, но тем не менее "и такой вариант возможен" (©).
С подробным описанием приведенных выше проблем можно ознакомиться в Вирусной энциклопедии.

Все-таки не лишним будет заметить, что далеко не все антивирусы страдают таким параноидальным отношением к pchook.dll. К примеру, Avira AntiVir и Avast (между прочим, абсолютно бесплатные для персонального использования) совершенно лояльно относятся к этому файлу.

Проверка Internet Explorer

Простенький тест для проверки восстановления паролей именно в Internet Explorer.

  1. Откройте эту страницу в Internet Explorer.
  2. Измените пароль в поле ввода "Пароль для проверки", расположенном ниже, на любую понравившуюся Вам строку.
  3. Запустите Password Cracker.
  4. Установите галочку "Восстанавливать в Internet Explorer" в настройках Password Cracker (меню кнопки "Настройки").
  5. Активизируйте элемент ввода пароля, расположенный ниже (просто кликните по нему мышкой).
Пароль, набранный в элементе ввода расположенном ниже, должен отобразиться в поле "Пароль" окна Password Cracker. Если программе удастся, то восстановленный пароль также будет отображаться во всплывающей подсказке к полю ввода пароля.

Локализация Password Cracker

Password Cracker поддерживает внешнюю локализацию пользовательского интерфейса (перевода на иностранный язык). Для выбора другого языка интерфейса достаточно нажать на кнопку "Options" и выбрать любой язык из подменю "Language" ("Язык" в русской версии).

Английский язык поддерживается непосредственно Password Cracker, все остальные языки пользовательского интерфейса располагаются в папке программы в файлах с расширением *.lng. Например, русский перевод расположен в файле Russian.lng.
Вы может скопировать Russian.lng в файл с другим названием, и перевести все названия в этом файле, находящиеся справа от знака "=" (равно). Редактировать lng-файлы можно в любом текстовом редакторе, например в обычном Блокнот из дистрибутива Windows. Пожалуйста, если Вас не затруднит, пришлите мне Ваш файл перевода по электронной почте carc@rambler.ru, и я с моими благодарностями и Вашими копирайтами (©) включу Ваш файл перевода в официальный дистрибутив Password Cracker.

По Вашему желанию в "спасибках" за перевод может быть указана какая-либо гиперссылка на Вас (Ваш сайт, Ваш блог, Ваша аська... что угодно!).
