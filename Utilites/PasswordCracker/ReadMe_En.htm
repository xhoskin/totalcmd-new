Password Cracker - the tool for restoring forgotten passwords.

Installation

Password Cracker runs with Windows NT4/95/98/ME/2000/XP/Vista and does not require any installation. Simply create a new folder and unpack all files from zip-archive.

Localization

  1. Copy any file with extension "*.lng" to new file. E.g. copy "Russian.lng" to "French.lng" (for french localization).
  2. Translate strings right from symbol "=" in file with extension "*.lng" (here in the file "French.lng").
  3. Restart Password Cracker.
  4. Please, send me Your language file on e-mail carc@rambler.ru. I include Your lng-file to official Password Cracker distributiv.
