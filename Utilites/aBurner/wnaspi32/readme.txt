Русский
=======
 * Проверьте наличие сторонней WnASPI32.DLL в папке \System32\. В случае обнаружения удалите его.
 * Скопируйте WnASPI32.DLL в папку aBurner для использования режима ASPI.

English
=======
 * Check the presence of third-party WNASPI32.DLL in the \System32\ folder. In this case delete this file.
 * Copying WnASPI32.DLL to aBurner folder for using ASPI mode.
