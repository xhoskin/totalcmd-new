HLP/MVB Plugin for Total Commander

Version 1.0.2.0

Packer plugin that allows to unpack help files in HLP and MVB formats.

Author:Konstantin Vlasov, 2011
Homepage:http://flint-inc.ru/
E-mail:support@flint_inc.ru

Distributed under Artistic License 2.0.

The plugin is based on the source code of the HelpDeco tool by Manfred Winterhoff, distributed as Freeware.
The tool's homepage is unavailable; development continued at SourceForge.net under the GPLv2 license.

System requirements for 32-bit version:
Processor with SSE2 (Intel Pentium 4, AMD Athlon 64 and higher)
Windows 2000 and higher
Total Commander 7.50–8.0 x32

System requirements for 64-bit version:
Processor with 64-bit instrustion set (Intel Pentium 4, AMD Athlon 64 and higher)
Windows XP x64 and higher
Total Commander 8.0 x64

Installation

For installing the plugin just enter its archive in Total Commander and agree to install it. If automatic installation is turned off in your TC, see instructions in the Total Commander help file.

After the installation you will be able to open HLP and MVB files as archives by Ctrl+PageDown hotkey, or unpack them by Alt+F9 hotkey. Packing into archive (i. e. help file compilation) is not supported.
