Новое в версии 6.0.3:
  - удалены все настройки, которые теперь можно
    установить из диалогов самого TC 6.0;
  - добавлены все скрытые опции, имеющие отношение
    к настройкам режимов работы TC 6.0;   
  - поддержка переменных окружения (в том числе и 
    %commander_path%) в командной строке и в путях, 
    используемых в INI-файлах TC;
  - всплывающие подсказки теперь содержат справочную
    информацию о формате записей в ini-файле;
  - предпросмотр пользовательских библиотек с картинками
    для кнопок панели дисков;
  - при отключении опций "Загружать значки/картинки..."
    показывает оригинальные значки/картинки (не работает
    для картинок если ресурсы totacmd.exe упакованы).
   
  Совместимость с TC 6.5x (сохранность новых значений): 
  - для SortUpper;
  - для InplaceRenamePath.

  Финальная версия укомплектована небольшой библиотечкой 
  с симпатичным набором картинок  для  замены стандартных 
  картинок кнопок дисков (автор библиотеки мне неизвестен)
  Файл библиотеки: MG_DriveBmp.dll   

  SR1:
  - устранен источник ошибки "...Decrypt.dll not found!"
  - по-умолчанию "пузырьковый" стиль всплывающих посказок
    в XP теперь отключен (есть проблемы с его отображением).
    Можете попробовать включить его путем добавления 
    в TweakTC.ini строки BalloonStyleTT=1 ;
  - более корректное отображение усеченных длинных путей.

  SR2:
  - теперь кнопка "Применить" ведет себя так, как это
    обычно принято в диалоговых окнах;
  - исправлена ошибка, из-за которой при запуске программы
    не из TC некорректно  обрабатывались пути, содержавшие
    в себе %commander_path%;
  - теперь иконки в окне предпросмотра пользовательских
    библиотек сгруппированы попарно (32х32 и 16х16);
  - превью для стандартных картинок кнопок дисков теперь
    видны независимо от упаковки ресурсов totacmd.exe;   
  - превью для картинок к кнопкам вызова меню директорий
    и истории посещенных директорий больше не показываются
    (причина: они НЕ используются текущей версией TC).

  SR3:
  - добавлена опция указания расположения регистрационного
    ключевого файла TC (wincmd.key или tcmdkey.zip), 
    отличного от стандартного (в рабочей директории TC);
  - в диалогах FTP добавлена опция для импорта/экспорта 
    установок файервола / прокси сервера;
  - при запуске TTC из-под Тотала считывается командная
    строка самого Тотала и уже из нее берутся (если 
    присутствуют!) пути к ини-файлам Тотал Коммандера; 
  - увеличена скорость записи-чтения ини-файлов в NT/XP
    за счет работы на уровне секций ини-файлов;
  - список для пометки дисков "как CDROM" теперь содержит
    полный перечень дисков (обход проблемы с внешними CDROM);
  - исправлены некоторые обнаруженные несоответствия надписей 
    на элементах управления их рельным функциям;
  - подсказки для списков выбора дисков теперь содержат
    строку с именами уже выбраных дисков;
  - добавлены кнопки "больше / меньше" для числовых полей.

1. Назначение программы Tweak Total Commander (далее TweakTC)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
    TweakTC  предназначен  для  тонкой  настройки Total Commander
версий  6.0  -  6.03a  без  редактирования вручную его INI-файлов
(обычно  wincmd.ini  и  wcx_ftp.ini).  При использовании TC более
ранних  нежели 6.03 версий некоторые опции работать не будут , но
и  не  могут  помешать  работе  TC.  TweakTC определяет положение
INI-файлов   такими  же  способами,  как  и  TC,  т.е.  если  при
инсталляции  TC  или позже Вы изменили их размещение относительно
типового  (в  папке Windows), то это будет обнаружено. Кроме того
Вы   можете   указать  путь  к  INI-файлам  в  командной  строке,
аналогично  тому,  как  это  может быть сделано в самом TC (можно
просто непосредственно скопировать эти параметры).

    Автор  TweakTC  не  является  экспертом по части настроек TC.
Если  Вам нужны дополнительные разъяснения по опциям, обращайтесь
на  официальный  сайт  www.ghisler.com,  шлите  письма  Christian
Ghisler  <support@ghisler.com>.  Для  русскоязычных пользователей
очень рекомендую  посетить  http://wincmd.ru/ и, особенно,  форум 
пользователей Тотал Коммандер на http://forum.wincmd.ru/. 


2. Установка программы
~~~~~~~~~~~~~~~~~~~~~~

Для установки программы TweakTC на Вашем компьютере:

  а) распаковать ZIP файл;
  б) скопировать его содержимое в удобное для вас место;
  в) создайте ярлык к программе;



3. Требования к системе
~~~~~~~~~~~~~~~~~~~~~~~

    Для   работы   программы   на  вашем  компьютере  нужен  файл
msvbvm50.dll, стандартно поставляемый в комплекте MS-WINDOWS 98 и
выше (исключение - Windows  2003). При его отсутствии в системных
папках  следует  скопировать  с другого компьютера или извлечь из
дистрибутива Windows98+.
    При  запуске  на некоторых компьютерах может возникать ошибка
типа   "Unknown...".   Это   возможно  при  отсутствии  некоторых
системных  библиотек  в операционной системе. Решить эту проблему
проще  всего  установкой  какой-либо  программы, разработанной на
Visual  Basic  (затем  ее можно удалить, нужные файлы останутся в
системе).  Еще  лучше  поискать  в  сети или в вашей коллекции CD
файлы   VBRUN5*.exe   или  VBRUN6*.exe.  Запустите  их  на  Вашем
компьютере  (кстати  VBRUN5*.exe содержит в себе и msvbvm50.dll).
Этого должно быть достаточно.

4. Использование программы
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    Надеемся  что  освоение TweakTC не затруднит Вас. :-) 
    Если Вы  указываете  при  запуске  TC имена INI файлов в его
командной  строке,  Вы  можете  просто перенести эти параметры в
командную строку TweakTC, например, указав их в свойствах ярлыка.
    Для  записи  изменений,  внесенных с помощью  TweakTC следует 
использовать кнопку Применить или выйти из программы нажатием  ОК.
Для того чтобы изменения вступили в силу следует перезагрузить TC.
    При FTP экспорте/импорте соединения с совпадающими именами не
перезаписываются  (только  переименовываются), совпадающие ссылки
пропускаются.
    Если Вы обнаружили, что TweakTC никак не влияет на работу TC,
проверьте соответствие определенных программой путей к INI файлам
(для  этого предусмотрена кнопка 'INI') тем путям, что использует
TC (они указаны в диалоге 'О программе' самого TC).
    УЧТИТЕ: ввод значения по умолчанию для какого-либо параметра
приводит к удалению параметра из INI файла (за ненадобностью).
    ВНИМАНИЕ: Если Ваш компьютер использует нестандартные размеры
шрифтов   экрана   и   при  этом  надписи  в  программе  искажены,
воспользуйтесь  горячей  клавишей  Сtrl+F.  В появившемся списке
выберите подходящий размер шрифта интерфейса.
 
5. Удаление программы
~~~~~~~~~~~~~~~~~~~~~~

    Для удаления программы с Вашего компьютера достаточно удалить
файл  TweakTC.exe  и  его ярлык, если он был Вами создан, а также
другие файлы из дистрибутива, если они Вами были использованы.
    Никаких записей в реестр программа при работе не вносит.
