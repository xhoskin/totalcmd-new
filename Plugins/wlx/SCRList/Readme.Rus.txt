Scrlist 1.0 - плагин для Total Commander
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
(c) Храмов Евгений hram-ov@mail.ru

    Позволяет просматривать хранители экрана(файлы с расширением *.scr) по F3 или Ctrl+Q   

Инсталляция:
~~~~~~~~~~~
1. Распакуйте Scrlist.wcx в любой каталог
   (например туда, где вы храните другие плагины).
2. В Total Commander'е выберите
   Настройки|Плагины|Плагины просмотра (*.WLX).
3. Нажмите 'Добавить'; 
4. Выберите файл scrlist.wlx и жмите 'Ok'
5. Жмите 'OK'. Давите 'OK'.

Использование:
~~~~~~~~~~~~~
1. Выберите в TC файл хранителя экрна;
2. Для просмотра в отдельном окне нажмите F3 для просмотра в боковой панели нажмите Ctrl+Q;
3. Для закрытия отдельного окна нажмите Esc, а для выхода из режима быстрого просмотра еще раз нажмите Ctrl+Q;
